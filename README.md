<h1 align="center">Documentation</h1>

This repository contains all the raw markdown files for all the company's project implementation documentation.

## Development

> [!IMPORTANT]
> Astral [uv](https://docs.astral.sh/uv/getting-started/installation/#installation-methods) must be installed.

1. Sync dependencies

```bash
uv sync
```

2. Run the documentation server locally at `https://localhost:7000`.

```bash
uv run zensical serve -a 0.0.0.0:7000
```
